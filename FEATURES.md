# 機能一覧

## 🎨 ユーザーインターフェース

### レスポンシブデザイン
- デスクトップ、タブレット、モバイルに対応
- モダンで直感的なUI
- Tailwind CSSによる美しいデザイン

### サイドバーナビゲーション
- 折りたたみ可能なサイドバー
- アイコンとラベル付きメニュー
- アクティブページのハイライト
- ユーザー情報とログアウトボタン

## 🔐 認証・セキュリティ

### ログイン機能
- ユーザー名とパスワードによる認証
- JWT（JSON Web Token）による安全な認証
- トークンの自動更新と管理
- セキュアなパスワードハッシュ（bcrypt）

### 権限管理
- ロールベースのアクセス制御
- 管理者（admin）と一般ユーザー（user）
- 権限に応じた機能制限
- 自分自身のアカウントは削除不可

## 📊 ダッシュボード

### 統計カード
- 総ユーザー数
- 総コンテンツ数
- 公開済みコンテンツ数
- 総閲覧数

### グラフとチャート
- **折れ線グラフ**: 過去7日間のコンテンツ作成数
- **円グラフ**: ステータス別コンテンツ分布
- **棒グラフ**: カテゴリ別コンテンツ数
- **ランキング**: 人気コンテンツTop 10（閲覧数順）

### リアルタイム更新
- ページをリフレッシュすると最新データを取得
- アクティビティログの記録

## 👥 ユーザー管理

### ユーザー一覧
- テーブル形式での表示
- ユーザー名、メール、権限、ステータスなどの情報
- アバター表示（イニシャル）
- 作成日の表示

### 検索・フィルタリング
- リアルタイム検索（ユーザー名、メール、氏名）
- ロールによるフィルタリング
- ステータスによるフィルタリング

### CRUD操作（管理者のみ）
- **作成**: 新規ユーザーの追加
- **読取**: ユーザー情報の表示
- **更新**: ユーザー情報の編集
- **削除**: ユーザーの削除
- 自分自身のアカウントは削除不可

### ユーザーフィールド
- ユーザー名（一意）
- メールアドレス（一意）
- パスワード
- 氏名（任意）
- 権限（管理者 / ユーザー）
- ステータス（有効 / 無効）

## 📝 コンテンツ管理

### コンテンツ一覧
- テーブル形式での表示
- タイトル、スラッグ、ステータス、カテゴリなど
- 作成者名と作成日
- 閲覧数の表示

### 検索・フィルタリング
- リアルタイム検索（タイトル、本文）
- ステータスによるフィルタリング
  - 下書き（draft）
  - 公開（published）
  - アーカイブ（archived）
- カテゴリによるフィルタリング

### コンテンツ作成・編集
- シンプルなフォームエディター
- **タイトル**: 記事のタイトル
- **スラッグ**: URLフレンドリーな識別子（自動生成）
- **カテゴリ**: コンテンツの分類
- **ステータス**: 下書き、公開、アーカイブ
- **本文**: テキストエリアでの編集

### 権限管理
- 自分が作成したコンテンツは編集・削除可能
- 管理者はすべてのコンテンツを編集・削除可能
- 公開ステータス変更時に公開日を自動記録

### 閲覧数トラッキング
- コンテンツごとの閲覧数を記録
- ダッシュボードで人気コンテンツを表示

## ⚙️ 設定

### アカウント情報

- メールアドレス
- 氏名
- 権限

- `/api/content/*` - コンテンツ管理
- `/api/analytics/*` - 分析・統計

### エラーハンドリング
- 適切なHTTPステータスコード
- わかりやすいエラーメッセージ
- バリデーションエラーの詳細

- CORS設定
- SQLインジェクション対策
